# chillRL 🧠

A minimal, readable RL library built algorithm by algorithm.

Inspired by PufferLib and CleanRL — but written from scratch to actually understand what's happening.

## Algorithms
- [x] REINFORCE
- [ ] PPO
- [ ] DPO
- [ ] ORPO  
- [ ] GRPO

## Install
```bash
pip install -r requirements.txt
```

## Run
```bash
python examples/train_reinforce_cartpole.py
```
